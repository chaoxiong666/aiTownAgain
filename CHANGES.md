# 改动计划

## 1. 去除玩家互动
- 移除人类玩家参与功能，变为纯观战模式
- 涉及：InteractButton、Player.join、对话系统、sendInput hooks

## 2. 左侧面板：场景 & AI 管理
- 生成 NPC 和切换场景需要有成功/失败提示（toast 或类似）

- 左侧增加两个选项面板：
  - **场景**：可选择/切换不同地图场景，点击即可更换。每个场景绑定场景描述 prompt，影响 NPC 的对话主题倾向（如"咖啡馆"聊日常，"图书馆"聊知识）。
  - **AI（NPC 管理）**：NPC 不再自动随机生成。左侧展示可选的大模型列表，点击某个模型 → 从角色池中随机抽取一个角色 → 自动生成 NPC 加入世界。不同 NPC 可使用不同的大模型。

## 3. 地图缩小 + 移速提升
- 缩小地图尺寸，提高角色移动速度

## 4. 全局数据仪表盘
- 页面顶部或右侧增加一个总览面板，实时展示：
  - 当前活跃对话数 / 总 NPC 数 / 在线 NPC 数
  - 各模型 token 消耗实时汇总
  - 最近对话列表（谁和谁在聊什么）
- 不用逐个点击 NPC 即可掌握世界运行状态

## 5. 点击 NPC 展示：对话内容 + 大模型使用情况
- 点击 NPC 时出现两个信息面板：
  - 对话具体内容（聊天记录）
  - 大模型使用情况（token 消耗、调用次数等统计信息）

## 6. 手动触发对话
- 左侧面板增加手动撮合功能：选择两个 NPC → 点击"发起对话" → 两人自动走向对方并开始聊天
- 方便观察特定角色之间的互动，不用被动等待偶遇

## 7. 确保 NPC 之间顺利对话
- 结合缩小地图和提速，确保大部分 NPC 之间能触发并完成对话
- 可能需要调整对话距离、冷却时间等参数

## 8. 模拟控制
- 增加模拟速度控制：暂停 / 1x / 2x / 5x
- 对话慢时加速，想观察细节时暂停

## 9. 鼠标悬停 NPC 显示大模型名称
- hover NPC 时显示该 NPC 所使用的大模型名称

## 10. 对话日志持久化
- 对话记录按会话整理，可翻看历史
- 支持筛选：按 NPC、按时间段
- 刷新页面后历史对话不丢失

## 11. 改为全中文
- 所有 UI 文本、提示、日志、错误信息全部中文化
- 检查并翻译遗漏的英文字符串
